# flappy_mike

This project is a *Flappy Bird* clone, nicknamed *FlappyMike* (until a better name is found ofc), implemented on the CY8CKIT-059 PSoC 5LP Prototyping Kit.

The game's goal is to navigate a "bird" through a series of pipes by "flapping." The game is displayed on an OLED Click module, and the user input (the "flap" command) is provided by tapping or making a sound near the Mic 2 Click module, which is compared against a set threshold.

## Core Hardware

- **Main Board**: [Cypress CY8CKIT-059 PSoC 5LP Prototyping Kit](./doc/cykit_059_schematics.pdf)
- **Adapter Shield**: [Cypress Click Shield](./doc/cypress_schematic.pdf)
- **Display**: OLED C Click Module
- **Input**: Mic 2 Click Module
